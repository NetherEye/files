NodeJs下载：https://nodejs.org/en/download/
Node中文网址：http://nodejs.cn/ (建议你在这里下载，速度会快很多)
node环境配置：https://www.cnblogs.com/zhouyu2017/p/6485265.html
1.创建两个文件夹【node_global】及【node_cache】
2.npm config set prefix "D:\Develop\nodejs\node_global"
npm config set cache "D:\Develop\nodejs\node_cache"
3.进入环境变量对话框，在【系统变量】下新建【NODE_PATH】，输入【D:\Develop\nodejs\node_global\node_modules】，
将【用户变量】下的【Path】修改为【D:\Develop\nodejs\node_global】
